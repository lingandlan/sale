### 优先级: P2 (中等)

#### BUG-001: 退出登录功能失效

| 项目 | 内容 |
|-----|------|
| **页面** | 全局（侧边栏） |
| **问题描述** | 点击侧边栏「退出登录」按钮后，页面未跳转到登录页，仍停留在当前页面 |
| **URL** | http://localhost:5175/dashboard (点击前) |
| **复现步骤** | 1. 登录系统 2. 点击侧边栏「退出登录」按钮 |
| **预期结果** | 弹出确认对话框，点击确定后跳转到 /login |
| **实际结果** | 弹出确认对话框，但点击确定后 URL 仍为 /dashboard |
| **代码位置** | shop-pc/src/layouts/Sidebar.vue:147-159 |
| **状态** | ❌ 待修复 |

**问题代码**:
```javascript
const handleLogout = () => {
  ElMessageBox.confirm('确定要退出登录吗？', '提示', {
    confirmButtonText: '确定',
    cancelButtonText: '取消',
    type: 'warning'
  }).then(() => {
    localStorage.removeItem('access_token')
    localStorage.removeItem('refresh_token')
    router.push('/login')
  }).catch(() => {
    // 取消操作
  })
}
```

**可能原因**: 
- ElMessageBox.confirm 的 `.then()` 回调未正确执行
- 或者 `router.push('/login')` 被路由守卫拦截
