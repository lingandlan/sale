## 🔧 修复建议

### 1. BUG-001 到 BUG-005 (5个页面空白)

这5个页面可能是：
- 路由未正确配置
- 组件未正确导出
- 组件内部有 JS 错误导致白屏

**建议检查**:
- 检查 `router/index.ts` 中的路由配置
- 检查浏览器控制台是否有 JS 错误
- 检查组件是否正确 import

### 2. BUG-006 (退出登录失效)

**建议检查**:
- 检查 `logout` 函数是否正确调用 API
- 检查是否正确清除 token/localStorage
- 检查是否正确调用 `router.push('/login')`
