# 太积堂系统页面设计

## 已完成页面

### 1. 登录页面
- 手机号+密码/短信验证码登录
- 品牌展示区域（左侧红色背景+金色Logo）
- 用户协议提示

### 2. B端充值申请发起页面
- 充值中心选择
- 积分计算（基础1:1 + 返还比例，向上取整）
- 上月商城净消费显示
- 返还比例规则说明（≥10万返还2%，<10万返还1%）
- 付款凭证（银行流水单号或截图，至少填一项）
- 当前中心余额显示
- 备注输入（可选）

### 3. B端充值审批列表页面
- Tab切换（待审批/已审批）
- 筛选功能（申请日期范围 + 充值中心）
- 数据表格（充值中心、充值金额、预计积分、申请人、申请时间、状态、操作）
- 批量操作（批量通过/批量拒绝）
- 分页功能

## 文件说明

所有页面设计文件使用 `.pen` 扩展名，可以通过 Pencil 编辑器打开查看和编辑。
