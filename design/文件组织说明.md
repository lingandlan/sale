# 太积堂系统页面设计 - 文件组织

## 文件结构

```
design/
├── taijido-design-system.pen       # Design System 主文件
├── DESIGN-SYSTEM.md                 # Design System 文档说明
├── 文件组织说明.md                   # 本文件
├── logo/
│   └── logo_1.png                   # 太积堂品牌 Logo
└── pages/
    ├── 01-登录页.png                 # 登录页面预览图
    ├── 02-B端充值申请.png            # B端充值申请发起页面预览图
    ├── 03-B端充值审批列表.png        # B端充值审批列表页面预览图
    └── README.md                     # 页面功能说明
```

## 文件命名规范

- PNG预览图使用数字前缀：`01-`, `02-`, `03-` 等
- 清晰描述页面功能
- 使用 `.png` 扩展名用于预览图
- 使用 `.pen` 扩展名用于源文件（Pencil编辑器格式）
